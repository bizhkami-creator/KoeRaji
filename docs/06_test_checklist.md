# 06_テストチェックリスト (Test Checklist)

## Day 1-6 基本機能
- [x] アプリが正常に起動すること
- [x] フォルダ選択・再帰的ファイル検索ができる
- [x] 番組情報（局・番組名・日付）が正しくパースされる
- [x] 音声ファイルの再生・一時停止・停止ができる
- [x] 各操作に対してTTS（音声読み上げ）のフィードバックがある
- [x] 検索欄によるリアルタイム絞り込みが動く

## Day 7 検索強化 & 音声操作
- [x] マイク権限のリクエストが表示され、許可できる
- [x] 「再生」「停止」などの音声コマンドが機能する
- [x] 複数キーワード（例：「TBS 荻上」）でAND検索ができる
- [x] 多様な日付形式で正しく検索できる

## Day 8 自動選択機能
- [x] 検索結果が1件のときに自動で選択される
- [x] 自動選択時に番組情報が読み上げられる

## Day 9 音声操作の高度化
- [x] 「○○を再生」という音声コマンドで検索・再生ができる
- [x] 音声認識開始時に再生が一時停止する
- [x] 検索系コマンドのTTS応答後、再生が自動再開する
- [x] 多様な音声検索コマンド（～を検索、～を調べて等）が動作する
- [x] 「おととい」「3日前」「先週の月曜日」等の相対日付で検索ができる

## Day 10 再生ナビゲーション & アナウンス
- [x] 「前へ」「次へ」ボタンで音声が切り替わる
- [x] 音声コマンド「次」「前」で再生が切り替わる
- [x] リストの端でループ動作（最後→最初、最初→最後）が機能する
- [x] 未選択状態で「再生」と言い、1件目が自動選択・再生される
- [x] 再生開始直前に「局名、番組名、日付を再生します」とフルで読み上げられる
- [x] TTSの読み上げが完全に終わってから音声再生が始まる

## Day 11 シーク操作 (早送り・巻き戻し)
- [x] 「10秒戻る」「30秒進む」ボタンで再生位置が移動する
- [x] 音声コマンド「早送り」「巻き戻し」「戻る」「進む」で操作ができる
- [x] 音声操作時、案内（〇〇秒進みました）の後に正しく再生が再開される
- [x] 0秒未満やファイル終端を超えるシークをしてもエラーにならない
- [x] 音声ファイル未選択時にシーク操作をすると適切にエラー案内される
