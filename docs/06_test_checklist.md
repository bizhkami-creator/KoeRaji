# 06_テストチェックリスト (Test Checklist)

## Day 1-6 基本機能
- [x] アプリが正常に起動すること
- [x] フォルダ選択・再帰的ファイル検索ができる
- [x] 番組情報（局・番組名・日付）が正しくパースされる
- [x] 音声ファイルの再生・一時停止・停止ができる

## Day 7 検索強化 & 音声操作
- [x] マイク権限のリクエストが表示され、許可できる
- [x] 「再生」「停止」などの音声コマンドが機能する
- [x] 複数キーワード（例：「TBS 荻上」）でAND検索ができる
- [x] 「と」区切りの入力（例：「TBSと荻上」）でAND検索ができる
- [x] 「6月15日」や「6/15」などの日付形式で正しく検索できる
- [x] 全角英数字（例：６月１０日）でも正しく検索できる
- [x] 「〜を探して」という音声コマンドで複数条件の絞り込みができる
